Adoty Plataforma para adoção de animais

Projeto desenvolvido com Angular 9, Node.js, Typescript, Rest API e banco de dados MySQL.


O sistema conta com as funcionalidades de: Cadastro de ONGs, Animais e Adotantes, persistindo as informações em um banco de dados, Sistema de Login e Autenticação via JWT, Fluxo de adoção de animais, Visualização de detalhes de Animais e ONGs (informações e fotos) e Sistema de integração com APIs.
